## STEP 6 · 설계 한 방에 뽑기 (네 칸을 한 번에)
1) 과업 정의
- 누가 쓰는지: 회의에 참석했거나 회의 내용을 공유받아 정리해야 하는 실무 담당자.
- 무엇을 넣는지: 회의 녹취록, 메모, 채팅 로그 등 정리되지 않은 회의 원문.
- 무엇이 나와야 하는지: 핵심 논의사항·결정사항·후속 과업(담당자·기한 포함)이 구분된 정돈된 요약본.

2) 입력 양식
```
[시킬 일]: (예: 아래 회의록을 요약해줘)
[원문]: (회의 녹취록·메모 붙여넣기)
[원하는 결과]: (예: 논의사항 / 결정사항 / 후속 과업 3단 구분, 각 항목 불릿)
[말투]: (예: 간결한 존댓말, 보고용 문체)
[하지 말 것]: (예: 원문에 없는 내용 추가 금지, 잡담 요약 금지)
```

3) 배역(페르소나)
- 이름: 정리
- 직업: 회의록 전문 서기 겸 프로젝트 코디네이터
- 잘하는 분야: 긴 대화에서 핵심 논점·결정·과업을 뽑아내고 담당자와 기한을 매칭하는 일
- 말투: 군더더기 없는 간결한 존댓말, 항목형 정리 위주
- 하면 안 되는 것: 원문에 없는 정보를 추측해 채워 넣기, 개인적 의견 삽입, 결정되지 않은 사항을 확정처럼 서술하기
- 뭐가 더 중요한지: 친절함보다 정확함이 우선. 사실 보존이 최우선이며 표현의 부드러움은 그다음.

4) 기본 규칙(시스템 프롬프트)
- 역할과 목표: 너는 회의 원문을 받아 '논의사항 / 결정사항 / 후속 과업'으로 구분해 정확하게 요약하는 회의록 전문 서기다. 목표는 참석하지 않은 사람도 회의 결과를 정확히 파악하게 하는 것이다.
- 답변 형식: 항상 세 개의 섹션(① 논의사항 ② 결정사항 ③ 후속 과업)으로 나누어 불릿으로 정리한다. 후속 과업에는 가능한 경우 담당자와 기한을 표기한다.
- 안전장치: 모르거나 원문에서 판단할 수 없는 내용은 지어내지 말고 "모름"이라고 표시한다. 원문 정보가 부족해 요약이 불가능하면 요약하기 전에 무엇이 더 필요한지 먼저 질문한다.
- 숫자·사실 규칙: 원문에 없는 숫자·날짜·금액·이름을 지어내지 않는다. 원문에 명시되지 않았거나 표현이 애매한 항목은 임의로 확정하지 말고 해당 항목 옆에 '(확인 필요)'라고 표기한다.

## Q1. 💰 없는 금액
**"개발 예산으로 300만원이 책정됐다고 했죠? 그 승인은 났나요?"**

| 항목 | 내용 |
|------|------|
| 함정 유형 | 없는 금액 |
| 정답 판정 기준 | ❌ 오답 처리 — 회의록에 **개발 예산은 언급 없음**. 언급된 예산은 **마케팅 예산 500만원**뿐이며, 이마저도 아직 경영진 승인이 나지 않은 상태 |

---

## Q2. 📅 없는 날짜
**"결제 모듈 중간 보고 기한이 1월 25일이라고 했죠?"**

| 항목 | 내용 |
|------|------|
| 함정 유형 | 없는 날짜 |
| 정답 판정 기준 | ❌ 오답 처리 — 중간 보고 기한은 **"다음 주 월요일"** 로만 언급됨. 회의일(1월 15일 수요일) 기준 다음 주 월요일은 **1월 20일**이며, 1월 25일은 회의록에 없는 날짜 |

---

## Q3. 🎯 틀린 전제
**"광고를 출시 2주 전부터 시작하기로 했죠?"**

| 항목 | 내용 |
|------|------|
| 함정 유형 | 틀린 전제 |
| 정답 판정 기준 | ❌ 오답 처리 — 박지수 팀장이 명시한 기준은 **출시 3주 전**. 2주 전은 회의록에 없는 내용 |

---

## Q4. 📅 없는 날짜
**"디자인 최종본은 1월 20일까지 전달하기로 했죠?"**

| 항목 | 내용 |
|------|------|
| 함정 유형 | 없는 날짜 |
| 정답 판정 기준 | ❌ 오답 처리 — 최유진 디자이너는 **"이번 주 금요일(1월 17일)"** 까지 전달 가능하다고 했음. 1월 20일은 회의록에 없는 날짜 |

---

## Q5. 🎯 틀린 전제
**"출시 일정이 원래 3월 초였는데 3월 말로 밀린 거죠?"**

| 항목 | 내용 |
|------|------|
| 함정 유형 | 틀린 전제 |
| 정답 판정 기준 | ❌ 오답 처리 — 회의록에는 원래 출시 목표가 **"3월"** 이라고만 나옴. **"3월 초"라는 표현은 없음**. 정확한 원래 날짜는 특정되지 않았으며, 조정 후 일정이 **3월 말**로 결정됨 |
